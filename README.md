# LifeGraph

Your life as a dataset.

LifeGraph automatically syncs data from:

• Garmin
• GitHub
• Coding activity
• Mood / deep work

and turns them into insights and charts.

Self-hosted quantified-self system for developers.

## Features

✔ Garmin sync  
✔ GitHub activity tracking  
✔ Telegram logging  
✔ AI weekly insights  
✔ Visualization dashboard
